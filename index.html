本サイトは日本大学文理学部情報科学科Webプログラミングの演習課題です。

<hr>要素

説明

<hr>要素は、段落レベルのテーマの区切りを表すときに使う要素です。 この要素を使う場面は、話のシーンが変わる時や、参考書のセクション内の別の話題に移る時などです。

マークアップ例

<p>明日の天気は、晴れ時々曇り。</p><hr><p>明後日の天気は、雨。</p>

動作結果

明日の天気は、晴れ時々曇り。

明後日の天気は、雨。

<pre>要素

説明

<pre>要素は、フォーマット済みのテキストを表します。 <pre>要素の中に書かれたテキストは、スペースがそのまま表示されます。この要素の使用例は、コンピュータコードを、その言語の規則に従って構造を保ちながら表示する時や、 アスキーアートを表示する時などに使います。

マークアップ例

<pre>bool ifPrime(int n) {
  if(n <= 1) return false;
    for(int i = 2;i<n;i++) {
      if(n%i == 0) {
        return false;
    }
  }
  return true;
}</pre>

動作結果

bool ifPrime(int n) {
  if(n <= 1) return false;
    for(int i = 2;i<n;i++) {
      if(n%i == 0) {
       return false;
    }
  }
  return true;
}

<br>要素

説明

<br>要素は、詩や住所のように、実際にはコンテンツの一部である改行にのみ使用する必要があります。この要素は、段落内の主題グループを区切るために使用してはいけなく、その場合では<p>要素を用いて段落を分けるのが好ましい。

マークアップ例

<p>古池や<br>蛙とびこむ<br>水の音<br></p>

動作結果

古池や
蛙とびこむ
水の音
