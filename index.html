<hr>要素

説明

<hr>要素は、段落レベルのテーマの区切りを表すときに使う要素である。 使用例は、話のシーンが変わる時や、参考書のセクション内の別の話題に移る時などです。

マークアップ例

<p>明日の天気は、晴れ時々曇り。<hr>明後日の天気は、雨。</p>

動作結果

明日の天気は、晴れ時々曇り。

明後日の天気は、雨。



<pre>要素

説明

<pre>要素は、フォーマット済みのテキストを表します。 <pre>要素の中に書かれたテキストは、スペースがそのまま表示される。この要素の使用例は、コンピュータコードを、その言語の規則に従って構造を保ちながら表示する時や、 アスキーアートを表示する時などに使います。

マークアップ例

  <pre>bool ifPrime(int n) {
    if(n &lt;= 1) return false;
    for(int i = 2;i&lt;n;i++) {
      if(n%i == 0) {
         return false;
      }
    }
    return true;
  }</pre>

動作結果

bool ifPrime(int n) {
  if(n <= 1) return false;
  for(int i = 2;i<n;i++) {
    if(n%i == 0) {
       return false;
    }
  }
  return true;
}
